# Auth.md

This site is a public technical blog. AI agents are welcome to access all content without authentication.

## Agent Registration

No registration is required for standard read-only access. AI agents should identify themselves using a proper `User-Agent` header (e.g., `GPTBot`, `Claude-bot`).

## Protected Resources

Currently, there are no protected resources on this domain. All APIs listed in the [API Catalog](/.well-known/api-catalog) are public.
